# Sales Data Analysis Project

## Project Overview
This project analyzes a retail sales dataset to identify key business insights such as regional sales performance, category contribution, top-selling products, and monthly sales trends.

The goal of this project is to demonstrate data analysis skills using SQL and data visualization using Excel.

---

## Tools Used
- SQL Server
- Microsoft Excel
- GitHub

---

## Dataset
The dataset contains retail sales information including:

- Order Date
- Region
- Category
- Product Name
- Sales

---

## Analysis Performed

### 1. Sales by Region
SQL was used to calculate total sales by region.

**Insight:**
West region generated the highest revenue, followed by East, while South recorded the lowest sales.

---

### 2. Sales by Category
Sales were aggregated by product category.

**Insight:**
Technology generated the highest revenue, followed by Furniture, while Office Supplies contributed the least.

---

### 3. Top 10 Products by Sales
Top-performing products were identified based on total sales.

**Insight:**
Canon imageCLASS 2200 Advanced Copier generated the highest revenue among all products.

---

### 4. Monthly Sales Trend
Sales were analyzed over time to identify growth patterns.

**Insight:**
Sales fluctuate month-to-month but show stronger performance during later periods.
